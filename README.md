# FitZone Fitness Club

The FitZone Fitness Club app is a fully functional 6-page HTML/CSS website with rich aesthetics, local images, and a complete GitHub deployment.

## 🚀 Live Demo
- **Live URL**: [rohit714786.github.io/Fitness-Club](https://rohit714786.github.io/Fitness-Club/)

## 🛠️ Features & Implementation

### 🎨 Premium Aesthetics
- **Color Palette**: Dark mode (#0a0a0a) with vibrant orange-red (#ff5000) accents.
- **Typography**: `Bebas Neue` for headlines and `Inter` for body text.
- **Micro-animations**: Smooth hover transitions on all interactive elements (buttons, cards, nav items).
- **Responsive Architecture**: CSS Grid `auto-fit` layouts for all device sizes.

### 📄 Page Structure
1. **Home (`index.html`)**: Dynamic hero, live stats count, why choose us features, and class previews.
2. **About (`about.html`)**: Company story, mission values, and industry awards.
3. **Classes (`classes.html`)**: Full program grid with descriptions, levels, and a comprehensive weekly timetable.
4. **Trainers (`trainers.html`)**: Personal profiles for 6 expert coaches with certification highlights.
5. **Pricing (`pricing.html`)**: Three-tier membership system and categorized FAQ.
6. **Contact (`contact.html`)**: Modern form (with JS-free success state), location details, and active opening hours.
